Introduction to C Language

C is a high-level, general-purpose programming language that is widely used for system and application development. It is known for its efficiency and flexibility in low-level programming.

Basic Syntax

A simple C program structure looks like this:

#include 

int main() {
    printf("Hello, World!\n");
    return 0;
}

This program prints "Hello, World!" to the console using the printf() function.

Variables and Data Types

In C, variables are used to store data values. C requires you to declare the type of variable before using it. Some common data types include:

Common Data Types

  • int - Used for integers (e.g., 1, 2, -5)
  • float - Used for floating-point numbers (e.g., 3.14, -2.5)
  • char - Used for characters (e.g., 'a', 'b')

Example

int a = 10;
float b = 5.5;
char c = 'A';

Operators in C

Operators in C allow you to perform various operations on data, such as arithmetic, relational, and logical operations.

Arithmetic Operators

These operators perform basic arithmetic operations:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)

Example

int a = 5;
int b = 2;
int result = a + b;  // result is 7

Control Flow

Control flow statements in C manage the flow of the program's execution. Common control flow statements include if, else, and loops (e.g., for, while).

if-else Example

int a = 10;
if (a > 5) {
    printf("a is greater than 5");
} else {
    printf("a is less than or equal to 5");
}

Functions in C

Functions allow you to break down the code into reusable blocks. Functions in C are declared with a return type, followed by the function name and parameters.

Function Example

#include 

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    printf("%d\n", result);  // Output: 8
    return 0;
}

Arrays in C

Arrays store multiple values of the same type in a single variable. Arrays in C are fixed in size once declared.

Array Example

int numbers[5] = {1, 2, 3, 4, 5};
printf("%d\n", numbers[2]);  // Output: 3

Pointers in C

Pointers are variables that store the memory address of another variable. They are powerful tools for managing memory in C.

Pointer Example

int a = 10;
int *p = &a;
printf("%d\n", *p);  // Output: 10

Structures in C

Structures allow you to group different data types together into a single unit. A structure is defined using the struct keyword.

Structure Example

struct Person {
    char name[20];
    int age;
};

struct Person p1;
p1.age = 30;
strcpy(p1.name, "John");
printf("%s is %d years old.\n", p1.name, p1.age);

Advanced Topics in C

Advanced topics in C include file handling, dynamic memory allocation, and multi-threading.

Dynamic Memory Allocation Example

#include 

int main() {
    int *arr = (int*) malloc(5 * sizeof(int));
    for (int i = 0; i < 5; i++) {
        arr[i] = i * 10;
