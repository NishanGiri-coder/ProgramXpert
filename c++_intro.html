Introduction to C++ Language

C++ is an object-oriented programming language developed by Bjarne Stroustrup in the early 1980s. It is widely used for system and application software development.

Basic Syntax

A simple C++ program structure is as follows:

#include 

int main() {
  std::cout << "Hello, World!";
  return 0;
}

The program starts from the main() function, which outputs "Hello, World!" to the console using std::cout.

Variables and Data Types

In C++, variables are used to store data values. Each variable has a type that determines what kind of data it can store.

Common Data Types

  • int - Used for storing integers (e.g., 1, 2, -5)
  • float - Used for storing floating-point numbers (e.g., 3.14, -2.5)
  • char - Used for storing a single character (e.g., 'A', 'b')

Example

int a = 10;
float b = 5.5;
char c = 'A';

Operators in C++

Operators are symbols used to perform operations on variables and values. They include arithmetic, relational, and logical operators.

Arithmetic Operators

These operators perform basic arithmetic operations:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)

Example

int a = 5, b = 2;
int result = a + b; // result is 7

Control Flow

Control flow statements manage the order in which code executes. Common control flow statements include if, else, and loops (e.g., for, while).

if-else Example

int a = 10;
if (a > 5) {
  std::cout << "a is greater than 5";
} else {
  std::cout << "a is less than or equal to 5";
}

Functions in C++

Functions allow us to group code into reusable blocks, making programs more modular.

Function Declaration Example

int add(int a, int b) {
  return a + b;
}

int result = add(5, 3);
std::cout << result; // Output: 8

Arrays in C++

Arrays store multiple values of the same type in a single variable.

Array Example

int numbers[5] = {1, 2, 3, 4, 5};
std::cout << numbers[2]; // Output: 3

Pointers in C++

Pointers store the memory address of a variable, allowing for dynamic memory manipulation.

Pointer Example

int a = 10;
int *ptr = &a;
std::cout << *ptr; // Output: 10

Classes in C++

Classes are the building blocks of object-oriented programming in C++. They allow you to define your own data types that can have both properties and behaviors.

Class Example

class MyClass {
public:
  int myNumber;
  void display() {
    std::cout << myNumber;
  }
};
MyClass obj;
obj.myNumber = 5;
obj.display(); // Output: 5

Dynamic Memory Allocation

Dynamic memory allocation allows programs to allocate memory during runtime, using functions like new and delete.

Example of new

int* ptr = new int[5];
ptr[0] = 10;
delete[] ptr;

Advanced Topics in C++

Advanced topics include inheritance, polymorphism, and templates.

Inheritance Example

class Animal {
public:
  void speak() {
    std::cout << "Animal speaks";
  }
};

class Dog : public Animal {
public:
  void speak() {
    std::cout << "Dog barks";
  }
};
Dog d;
d.speak(); // Output: Dog barks
