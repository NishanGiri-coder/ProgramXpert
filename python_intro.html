Introduction to Python Language

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used for web development, automation, and data analysis.

Basic Syntax

A simple Python program structure is as follows:

print("Hello, World!")

The program starts by calling the print() function to display "Hello, World!" on the console.

Variables and Data Types

In Python, variables store data values. Python automatically infers the type of the variable.

Common Data Types

  • int - Used for integers (e.g., 1, 2, -5)
  • float - Used for floating-point numbers (e.g., 3.14, -2.5)
  • str - Used for strings (e.g., 'Hello', 'Python')

Example

a = 10
b = 5.5
c = "Hello"

Operators in Python

Operators in Python perform operations on variables and values. They include arithmetic, relational, and logical operators.

Arithmetic Operators

These operators perform basic arithmetic operations:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)

Example

a = 5
b = 2
result = a + b  # result is 7

Control Flow

Control flow statements in Python manage the execution order of the code. Common control flow statements include if, else, and loops (e.g., for, while).

if-else Example

a = 10
if a > 5:
    print("a is greater than 5")
else:
    print("a is less than or equal to 5")

Functions in Python

Functions in Python allow grouping reusable code into blocks for better modularity.

Function Example

def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # Output: 8

Lists in Python

Lists store multiple values of different types in a single variable. Lists are mutable.

List Example

numbers = [1, 2, 3, 4, 5]
print(numbers[2])  # Output: 3

Tuples in Python

Tuples are similar to lists but are immutable. They are often used for fixed collections of items.

Tuple Example

numbers = (1, 2, 3, 4, 5)
print(numbers[2])  # Output: 3

Dictionaries in Python

Dictionaries store key-value pairs. They are unordered and mutable.

Dictionary Example

person = {"name": "John", "age": 30}
print(person["name"])  # Output: John

Advanced Topics in Python

Advanced topics include object-oriented programming, decorators, and generators.

Class Example

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

person1 = Person("John", 30)
print(person1.name)  # Output: John
